# Projects

A collection of small front-end projects built with pure HTML, CSS, and JavaScript.

---

## Projects

### ✅ To Do List
A simple task manager to add, complete, and delete tasks.
- Add and remove tasks
- Mark tasks as complete
- Clean, minimal UI

### ⏱️ Stopwatch
A functional stopwatch with start, stop, and reset controls.
- Start / Stop / Reset
- Displays time in HH:MM:SS format

### 💬 Random Quote Generator
Displays a random quote on each click.
- Curated quote collection
- One-click to generate a new quote
